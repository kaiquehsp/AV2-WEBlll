## 🛠️ Stack Tecnológica

* **Linguagem:** Java 17 
* **Framework:** Spring Boot 2.6.3
* **Persistência:** Spring Data JPA / Hibernate
* **Banco de Dados:** H2 (In-Memory)

---

## 🚀 Como Executar o Projeto (VS Code)

Siga os passos abaixo para rodar a aplicação localmente:

1.  Certifique-se de que o **Java JDK 17** está instalado e configurado no seu ambiente.
2.  No VS Code, navegue até o diretório do projeto e abra o arquivo principal:
    `AV2-WEBlll\automanager\src\main\java\com\autobots\automanager/AutomanagerApplication.java`
3.  Clique na opção **"Run"**
